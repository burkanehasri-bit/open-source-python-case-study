# Open Source Software Lab — Python Case Study

Name: B Neha Sri

Registration Number: 24MIP10182

Course: Open Source Software

---

 Project Overview

This project presents a case study on Python as an open-source software.
It explains Python’s origin, licensing, ecosystem, Linux integration, and comparison with proprietary software.

---

 Objectives

* Understand Python's open-source philosophy
* Study different software licenses (MIT, GPL, PSF)
* Analyze Python’s role in Linux systems
* Explore real-world applications of Python

---

 Technologies Used

* Python 3.13
* VS Code
* GitHub
* Git Bash

---

 Project Structure

*  Case Study Report (PDF)
*  Shell Scripts (script1.sh – script5.sh)
*  Screenshots of execution

---

 Scripts Description

* **script1.sh** → Check Python installation
* **script2.sh** → Display system information
* **script3.sh** → Run a Python program
* **script4.sh** → Simulate system update
* **script5.sh** → Create a service file
